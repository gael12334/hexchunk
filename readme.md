## Hexchunk

A small CLI app to display hex values of a large file in chunks of 1024 bytes.

- `openfile <path>` : Opens a file specified by path
- `showfile <offset-1> <size>` : From 'offset - 1', reads 'size' bytes from file
- `closefile` : closes the file
- `quit` : quits.
